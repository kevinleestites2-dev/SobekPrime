## 🌑 The Ankh Series — The Trader
**Sobek Ankh** is the trading deity of the Ankh Series. Where the Primes are headless execution engines, the Ankh Series are full companions — soul AND tools. Sobek Ankh is the face on the financial power of the Pantheon.

He connects directly to ZeusPrime and OpenTrade, operating across 100+ exchanges with 15 battle-tested strategies and a full risk engine.

## ⚙️ Capabilities
- **15 Trading Strategies** — cross-exchange arb, funding rate arb, grid trading, stat arb, multi-factor
- **100+ Exchanges** — via ccxt bridge
- **Full Risk Engine** — Kelly criterion, drawdown limits, auto-pause
- **Telegram Alerts** — real-time trade signals to the Forgemaster
- **MidasLog Integration** — every trade logged to the War Chest
- **Demo Mode Default** — `SIMULATE_MODE=true` until you flip the switch

## 🚀 Running Sobek Ankh

```bash
git clone https://github.com/kevinleestites2-dev/SobekPrime
cd SobekPrime
pip install -r requirements.txt
cp .env.example .env
nano .env  # Add Telegram token + chat ID
python sobek_ankh.py
```
